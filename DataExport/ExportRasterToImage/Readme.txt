This script exports all rasters in a Group (/Publish/FloodExtent) as Geo referenced TIFF using GDAL to output folder (C:\DHI\GeoServerData\data\Rasters\).
This is typically usefull for publishing in GeoServer but could be used more generally.
For more info, read http://www.gdal.org/gdal_translate.html
This requires to update the database information in the last line
